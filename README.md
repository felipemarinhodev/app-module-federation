# App
Essa é uma aplicação que é voltada para o estudo de Micro-frontends, usando Module Federation.

_Essa aplicação é responsável por orquestrar os micro-frontends_

Essa aplicação é executada na porta ```10001```.

## Instalando
```
npm i
```

## Executar o código
```
 npm run dev
```
